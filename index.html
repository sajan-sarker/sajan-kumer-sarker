Projects

CyberSec-Reasoner – A Three-Stage Post-Trained Reasoning LLM for Vulnerability Analysis

  • Engineered a domain-specialized LLM based on Qwen 3.5 (4B) designed to perform explicit Chain-of-Thought (CoT) reasoning for cybersecurity tasks, such as automated CVE-to-CWE mapping.
  • Architected a three-stage post-training pipeline featuring Knowledge-Based SFT for domain expertise, Cold Start SFT for reasoning structure, and Reinforcement Learning via Group Relative Policy Optimization (GRPO) for logical refinement.
  • Developed a weighted hybrid reward system (Total Reward = 0.5 · Rcwe + 0.3 · Rformat + 0.2 · Rkeyword) to optimize for classification accuracy and semantic grounding, validating the model against a 1,000-sample CVE/CWE dataset using custom rule-based metrics.
  • Tools: PyTorch, Transformers, TRL, Hugging Face PEFT, LoRA, QLoRA, GRPO, Qwen 3.5, Wandb.
